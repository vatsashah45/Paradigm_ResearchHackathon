Challenge description:
Write a strategy prompt for an AI agent that negotiates resource splits against a baseline across 10 games.

Each game, two AI agents split a pool of books, hats, and balls with hidden valuations. Both agents see the full negotiation history, including all proposals, actions, and messages from every prior turn. Your strategy prompt guides your agent across 10 games, alternating who moves first. Check out the GitHub repo to learn more.
https://github.com/paradigmxyz/negotiation-challenge
